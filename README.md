# Install

## Windows
`xcopy onedarkpro.xml %APPDATA%\QtProject\qtcreator\styles`

## MacOS
`cp onedarkpro.xml ~/.config/QtProject/qtcreator/styles/`

## Linux
`cp onedarkpro.xml ~/.config/QtProject/qtcreator/styles/`
