# Install

## Windows
`xcopy riva.xml %APPDATA%\QtProject\qtcreator\styles`

## MacOS
`cp riva.xml ~/.config/QtProject/qtcreator/styles/`

## Linux
`cp riva.xml ~/.config/QtProject/qtcreator/styles/`
